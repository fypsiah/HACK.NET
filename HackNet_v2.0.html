HOW TO PLAY

🎯 OBJECTIVE

Infiltrate systems, bypass security layers and extract data before being fully traced. Every successful hack earns Data (currency) and XP to level up.

🔄 THE LOOP

  • Select an available target from the Targets tab
  • Complete minigames to breach each security layer
  • Keep Trace below 100% — hit it and the hack fails
  • Spend Data to upgrade tools in the Tools tab
  • Level up to unlock the next tier of targets

🧩 MINIGAMES

  • Timing Grid — click when bar enters green zone
  • Code Match — select symbols matching the sequence
  • Sequence — memorise then repeat the lit pattern
  • Path Trace — navigate to goal (WASD / arrows)
  • Wire Cut — cut the correct wire from the directive
  • Binary Decode — convert binary string to decimal

📡 TRACE METER

Every action fills trace. Mistakes and loud tools spike it faster. Stealth reduces it. At 100% the hack is blown and all rewards are lost.

🔧 TOOLS

Upgrade tools with Data. Higher-tier tools (Ghost, Exploit) require higher player levels. Each tool has a cooldown per hack — use them wisely.

📈 LEVELLING UP

Each level unlocks 2–3 new targets. Difficulty scales from Easy (LVL 1) to Void-tier (LVL 10). Higher targets have more layers, harder minigames, and faster trace.

🗺️ NETWORK MAP

The Network tab shows the global node map across 8 zones from Local Networks to the hidden GENESIS cluster. Hover nodes for stats.

💡 TIPS

  • Use Stealth early — it compounds across all layers
  • Binary Decode and Wire Cut are all-or-nothing
  • Ghost Protocol drops 25% trace instantly
  • Exploit Kit auto-clears a layer with zero trace
